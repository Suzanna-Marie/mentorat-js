# Mentorat

## Objectif
Créer un projet simple pour permettre d'apprendre Git de l'installation à la création de projet, GitHub et la structure d’un projet JavaScript.

## Comment lancer le projet
1. Ouvrir le fichier `index.html` dans un navigateur web.
2. Ouvrir la console du navigateur pour voir le message "Mentorat OK".
